Transmute

Converta arquivos de um formato para outro com facilidade.

Sobre os formatos

Documentos

  • PDF - Documento portátil, ideal para leitura universal
  • DOCX - Documento editável do Microsoft Word
  • DOC - Formato antigo do Word
  • TXT - Texto simples sem formatação
  • RTF - Texto formatado rico
  • ODT - Documento aberto do LibreOffice
  • HTML - Código de página web
  • Markdown (MD) - Texto com formatação simples

Áudio

  • MP3 - Compressão leve com boa qualidade
  • WAV - Áudio sem compressão com qualidade máxima
  • AAC - Áudio com compressão avançada
  • FLAC - Áudio sem perdas, alta qualidade
  • OGG - Formato aberto de áudio
  • M4A - Formato do Apple iTunes
  • WMA - Formato da Microsoft

Vídeo

  • MP4 - Compatível com a maioria dos dispositivos
  • MOV - Formato da Apple com alta qualidade
  • AVI - Vídeo com menos compressão e maior tamanho
  • WMV - Formato da Microsoft para vídeo
  • FLV - Vídeo para Flash Player
  • MKV - Contêiner moderno e popular
  • WEBM - Vídeo para web
  • MPEG - Formato clássico
  • 3GP - Vídeo para celulares antigos

Imagens

  • JPG/JPEG - Formato popular com compressão
  • PNG - Imagem com transparência e compressão sem perdas
  • GIF - Animações simples e suporte limitado de cores
  • BMP - Formato sem compressão
  • TIFF - Formato para alta qualidade e edição
  • WEBP - Formato moderno para web
  • SVG - Gráficos vetoriais escaláveis
